Bot_salvador discord

Este es mi primer bot hecho con la finalidad de aprender y matar tiempo,
el objetivo es que este bot sea capaz de reproducir musica de youtube.

Para usarlo necesitas instalar lo siguiente:
node.js: https://nodejs.org/en/
discord.js : https://discord.js.org/#/docs/main/stable/general/welcome

Recomiendo instalar
Git: https://git-scm.com/
Visual estudio: https://code.visualstudio.com/


Los archivos mas importantes que necesitais (ya que los demas se generan con node.js y discord.js)
son: server.js y config2.json

Para poner en macha el bot una vez creado en el apartado developers de discord y teniendo todo instalado es lo siguiente

Creamos una carpeta con el nombre que queramos, abrimos una terminal en ella y ejecutamos el siguiente comando:

		nmp init -y 

esto es para que se creen los archivos de node.js

Ahora para crear los archivos de discord.js

		install discord.js --save

Con esto tendremos los archivos de discord.js

Ahora pasamos los archivos server.js y config2.json en la carpeta que hemos creado junto a los demas archivos
que se han instalado.

Lo que queda es renombrar config2.json a config.json abrimos con un editor de texto o con el visual studio el archivo y remplazamo "Token"
por el token de nuestro bot.

Y por ultimo para iniciar el bot abrimos un terminal en la carpeta (o el mismo que estabamos usando)
Y escribimos node server.js y el bot de beria estar funcionando.
